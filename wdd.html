Design Meets Programming

Web Design Decal is organized into two sections: design and programming. In programming, we teach the necessary technical skills and languages needed to create websites, like HTML, CSS, and Javascript. And in the design section, we teach principles like spacing, typography, and color to help our students craft the best user experience.

I took the class in Fall 2014, became a TA Spring 2015, and officially started lecturing Fall 2015. Not only do I enjoy designing and coding my own websites, I think web dev is a really useful skill for anyone to learn. Teaching is one of the best ways to improve my own skill, and I knew there would be plenty of bright students to learn from. UC Berkeley's design curriculum isn't the best, and I wanted to help change that.
