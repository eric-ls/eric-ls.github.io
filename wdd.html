Web Design Decal

Instructor

I was the head instructor for UC Berkeley's Web Design Decal, which teaches beginners how to design and code static websites.

Web Design Decal is organized into two sections: design and programming. In programming, we teach the necessary technical skills and languages needed to create websites, like HTML, CSS, and Javascript. And in the design section, we teach principles like spacing, typography, and color to help our students craft the best user experience.

I took the class in Fall 2014, became a TA Spring 2015, and officially started lecturing Fall 2015. Not only do I enjoy designing and coding my own websites, I think web development is a really useful skill for anyone to learn.

We structure the curriculum so that even beginners without any coding experience can jump straight into web design. Our students have produced stellar work, and the course staff are really honored to receive over 600 applications per year, with an average course rating of 4.9/5!
