Replate

I work part-time for Replate, a nonprofit that aims to reduce food waste and hunger in the community.

Re-Plate collects food donations from businesses and hires drivers to transport these donations to nonprofits and homeless shelters.

I started working with Replate in 2016 as a project leader in Blueprint. As a project leader, I led a team of 4 developers to build a web app for Replate to manage, track, and automate food donations.

Some work I completed

  • Dashboard for businesses to manage multiple office locations, request food pickups, and see their past transactions
  • Website design, branding, and frontend code
  • Savings Calculator for potential customers to estimate their financial savings
  • Integrations with Invoiced API to allow online payments, Intercom API to facilitate our Operations team, Google Maps API to track office locations, and Onfleet API to automate tasks to our drivers.

In January 2018, Re-Plate received a $250k donation from the Pineapple Fund.
